# 服装提案秋冬版
![plus test](https://github.com/u7una/robosys2025/actions/workflows/plus.yml/badge.svg)
![test](https://github.com/u7una/robosys2025/actions/workflows/test.yml/badge.svg)
ロボットシステム学課題1

入力した**気温(℃ )** に応じてその日に適した服装を提案するプログラムです。さらに **天気(晴れ/雨/曇り/雪)** を追加で与えると天気に応じた追加アドバイスも表示します。

---

## 機能概要
この 'plus' コマンドは、

+ 気温に応じた基本の服装提案
+ 天気に応じた追加アドバイス
+ 数値以外を入力した場合のエラー処理

を行います。

---
## 使い方


### 実行方法
$chmod +x plus - 実行権限を付与
$./plus 

## 実行例

### 入力例

$ 気温:20

$ 天気(晴れ/雨/曇り/雪) :晴れ

### 出力例

$ 服装:薄めのジャケット

$ 追加事項:日中は日差しが熱くなるかも

---

## 気温ごとの服装指数(仕様)
| 気温 | 提案される服装 |
|------|----------------|
|20℃ 以上|薄めのジャケット|
|15~19℃ |薄めのセーター、カーディガン、秋冬コート|
|10~14℃ |厚手のセーターやスウェット|
|5~9℃ |ダウン・ウールのコート+手袋・マフラー|
|5℃ 未満|厚手のダウン+ヒートテックなどの防寒必須|

---

## 天気に応じた追加アドバイス

| 天気 | 追加事項 |
|------|-----------|
| 晴れ | 日中は日差しが熱くなるかも|
| 雨 | 傘と撥水加工のある上着がおすすめ |
| 曇り | 日差しがなく少し肌寒いかも |
| 雪 | ブーツや手袋、カイロがあると良い |
