# 服装提案秋冬版
![test](https://github.com/u7una/robosys2025/actions/workflows/test.yml/badge.svg)
ロボットシステム学課題1

入力した**気温(℃ )** に応じてその日に適した服装を提案するプログラムです。さらに **天気(晴れ/雨/曇り/雪)** を追加で与えると天気に応じた追加アドバイスも表示します。

---

## 機能概要
この 'clothes_advice' コマンドは、

+ 気温に応じた基本の服装提案
+ 天気に応じた追加アドバイス
+ 数値以外を入力した場合のエラー処理

を行います。

```bash

$chmod +x clothes_advice
```

## 実行例

```bash
$ ./clothes_advice 20 晴れ
服装:薄めのジャケット
追加事項:日中は日差しが熱くなるかも
```
## 仕様補足
天気の入力が省略された場合は服装のみを表示し、追加事項は出力されません。

---

## 気温ごとの服装指数(仕様)
| 気温 | 提案される服装 |
|------|----------------|
|20℃ 以上|薄めのジャケット|
|15~19℃ |薄めのセーター、カーディガン、秋冬コート|
|10~14℃ |厚手のセーターやスウェット|
|5~9℃ |ダウン・ウールのコート+手袋・マフラー|
|5℃ 未満|厚手のダウン+ヒートテックなどの防寒必須|

---

## 天気に応じた追加アドバイス

| 天気 | 追加事項 |
|------|-----------|
| 晴れ | 日中は日差しが熱くなるかも|
| 雨 | 傘と撥水加工のある上着がおすすめ |
| 曇り | 日差しがなく少し肌寒いかも |
| 雪 | ブーツや手袋、カイロがあると良い |
